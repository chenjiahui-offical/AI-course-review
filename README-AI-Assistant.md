# AI学习助手使用说明

## 功能特性

### 1. 智能问答
- 💬 针对当前作业内容的智能问答
- 🤖 基于DeepSeek API的强大AI能力
- 📚 自动获取页面上下文，提供更准确的回答

### 2. 代码解释
- 📝 点击代码块右上角的"询问AI"按钮
- 🔍 快速获取代码解释和分析
- 💡 理解算法实现细节

### 3. 对话历史
- 💾 自动保存最近10轮对话
- 🔄 支持连续对话，AI会记住上下文
- 🗑️ 可随时清空对话重新开始

### 4. 安全存储
- 🔐 API密钥安全存储在浏览器本地
- 🚫 不会上传到任何服务器
- 🔑 随时可以修改或删除

## 使用步骤

### 第一步：获取DeepSeek API密钥

1. 访问 [DeepSeek平台](https://platform.deepseek.com)
2. 注册/登录账号
3. 在控制台创建API密钥
4. 复制API密钥（格式：sk-...）

### 第二步：配置API密钥

1. 点击页面右下角的紫色机器人按钮
2. 点击设置图标（齿轮）
3. 粘贴你的API密钥
4. 点击"保存"

### 第三步：开始使用

#### 方式1：直接提问
1. 在输入框输入你的问题
2. 按Enter或点击发送按钮
3. 等待AI回复

#### 方式2：询问代码
1. 找到任意代码块
2. 鼠标悬停在代码块上
3. 点击右上角的"询问AI"按钮
4. AI会自动分析代码并回答

## 注意事项

1. **API费用**：DeepSeek API按使用量计费，请注意控制使用
2. **网络连接**：需要稳定的网络连接才能使用
3. **隐私保护**：不要在对话中输入敏感个人信息
4. **准确性**：AI回答仅供参考，重要内容请以课程资料为准

## 常见问题

### Q: API密钥存储在哪里？
A: 存储在浏览器的localStorage中，只在你的设备上，不会上传。

### Q: 如何删除API密钥？
A: 打开设置，清空输入框，点击保存即可。

### Q: AI回答不准确怎么办？
A: 可以尝试：
- 提供更详细的问题描述
- 清空对话重新开始
- 指定你想了解的具体方面

### Q: 为什么代码块没有"询问AI"按钮？
A: 刷新页面，确保JavaScript已正确加载。

### Q: 可以使用其他AI服务吗？
A: 当前版本仅支持DeepSeek API，未来可能支持更多服务。
