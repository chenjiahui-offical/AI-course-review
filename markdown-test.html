## BFS算法详解

BFS（广度优先搜索）是一种**图遍历算法**。

### 主要特点

- 使用队列数据结构
- 层次遍历
- 保证最短路径

### 时间复杂度

| 操作 | 复杂度 |
|------|--------|
| 遍历 | O(V+E) |
| 空间 | O(V) |

### 代码示例

```python
def bfs(graph, start):
    visited = set()
    queue = [start]
    
    while queue:
        vertex = queue.pop(0)
        if vertex not in visited:
            visited.add(vertex)
            queue.extend(graph[vertex])
    
    return visited
```

> 注意：BFS适用于无权图的最短路径问题。
